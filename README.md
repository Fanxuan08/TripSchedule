# Trip Schedule (出國旅遊規劃與分帳神器)

## 專案簡介

是一個用 C# Windows Forms 開發的旅遊小工具。我認為出去玩最麻煩的通常就是兩件事：「排行程」跟「算錢」。
所以這個程式把「MBTI 行程規劃 (J人/P人模式)」、「多幣別記帳」和「自動算誰欠誰錢」這三大功能包在一起。加上完整的存檔功能，是一套非常實用的出遊好幫手！

### 基礎功能

* **一鍵存檔與讀取 (JSON 紀錄)**
  * 不用擔心打半天的資料不見，所有旅程、花費和行程都可以存成 `TripData.json`。下次打開程式點「載入舊旅程」，就能直接還原。
* **J人 / P人 雙軌行程規劃**
  * **J人模式 (精細規劃)**：用傳統的表格把幾點幾分、去哪裡、搭什麼車寫得清清楚楚。
  * **P人模式 (隨性自由)**：只分早、中、晚三個大區塊，想到什麼想完成的行程就丟進去卡片牆，主打一個高自由度。
* **出國記帳與分錢**
  * 支援輸入外幣（例如日圓）和自訂匯率，系統會自動幫你換算成台幣。
  * 可以設定這筆錢是「全體平分」還是「指定某幾個人分」，再複雜的代墊都不怕。

### 進階創意擴充

* **自動結算分帳**
  * 出去玩大家輪流墊錢，最後算帳總是會一團亂。這個系統會在背後自動把所有人的「代墊」跟「應付」抵銷，直接告訴你「A 只要轉帳給 B 多少錢就好」，讓轉帳筆數降到最低，使所有跟錢有關的事情變得更方便、減少糾紛。
* **主辦人權限控管**
  * 系統有分身分。只有切換成「主辦人」才能點擊「儲存檔案」來覆寫存檔；如果是其他同行小夥伴，按鈕會被反灰禁用（只能看不能存），不怕資料被雷隊友弄壞。
* **行程防衝堂警告**
  * 在 J 人模式排行程時，如果新加的活動跟原本的時間重疊了（例如同一個時間有兩個行程），系統會立刻跳出防呆警告提醒你，但如果想塞再一起也是可以（柔性勸導）。
* **一鍵匯出旅遊小手冊與帳單**
  * 排好的行程可以直接匯出成 `.txt` 文字檔，變成方便在群組傳閱或用手機看的旅遊手冊。
  * 記帳明細也可以匯出成 `.csv` 檔，方便事後用 Excel 對帳，誰也別想賴帳。

### 3. 行程規劃 (P/J 雙模式)
點擊「開始行程規劃」，左上角可以切換天數。
* **P 模式**：輸入活動後，點擊「新增」就會變成卡片貼在對應的時段。
* **J 模式**：輸入完整的時間與地點，如果不小心時間重疊，系統會跳出警告。
